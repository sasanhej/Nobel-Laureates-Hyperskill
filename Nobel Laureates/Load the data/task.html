Theory

JSON (JavaScript Object Notation) is a file format to store data. Let's look at an example:

{"born_in":{"0":"Argentina","1":"Bosnia and Herzegovina"},"category":{"0":"Physiology or Medicine","1":"Literature"},"country":{"0":"Argentina","1":""},"date_of_birth":{"0":"8 October 1927","1":"9 October 1892"},"date_of_death":{"0":"24 March 2002","1":"13 March 1975"},"gender":{"0":"male","1":"male"},"link":{"0":"http:\/\/en.wikipedia.org\/wiki\/C%C3%A9sar_Milstein","1":"http:\/\/en.wikipedia.org\/wiki\/Ivo_Andric"},"name":{"0":"C\u00e9sar Milstein","1":"Ivo Andric *"},"place_of_birth":{"0":"Argentina","1":"Bosnia and Herzegovina"},"place_of_death":{"0":"Cambridge , England","1":"Belgrade, SR Serbia, SFR Yugoslavia (present-day Serbia)"},"text":{"0":"C\u00e9sar Milstein , Physiology or Medicine, 1984","1":"Ivo Andric *,  born in then  Austria\u2013Hungary , now Bosnia and Herzegovina , Literature, 1961"},"year":{"0":1984,"1":1961},"year_born":{"0":1927,"1":1892},"age_of_winning":{"0":57,"1":69},"born_in_corrected":{"0":"Other countries","1":"Other countries"}}

This file format is convenient when storing data, but presents a challenge to interpret it manually. The Python pandas library is one of the instruments that can parse data stored as JSON.

Description

In this project, you will explore data on the Nobel laureates from various years. You have a dataset containing several columns:

  • born_in — Nobel laureate's country of birth.
  • category — Nobel Prize category.
  • country — the country the Nobel laureate represented.
  • date_of_birth — the Nobel laureate's date of birth.
  • date_of_death — the Nobel laureate's date of death.
  • gender — Nobel laureate's gender.
  • link — a link to the wiki page containing the information about Nobel laureate.
  • name — the Nobel laureate's name.
  • place_of_birth — the Nobel laureate's place of birth.
  • place_of_death — the Nobel laureate's place of death.
  • year — the year Nobel laureate obtained the Nobel Prize.

We have automated the data download process in the .py file provided to you. However, if that is inconvenient, feel free to download the dataset: nobel_laureates.json.

Objectives

In this step, you need to load and explore the data.

  1. Load the dataset from the JSON file;
  2. Explore the data and check what the axes are, get the shape of the DataFrame, use .info() to check for the data types and whether there are any missing values;
  3. Use the .duplicated() method to check whether there are any duplicate rows. Print False if there are no duplicate rows, otherwise print True.
  4. Drop all the rows where the gender column does not contain any values. Hint: use the subset parameter of the .dropna() method in pandas.
  5. Re-index the DataFrame so that the index contains consecutive numbers. Do not forget to remove the old index column.

To complete this stage, output the results of the step 3. Then on a new line print a dictionary containing the first 20 rows of the DataFrame with the DataFrame method .to_dict() for the country and name columns.
